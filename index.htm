Use cases:

How a user accesses the site

  • User: Type login credentials into user name and password field
  • User: Press return
  • Site: Verify user submitted login credentials
  • Site: Grant or deny user accesss
  • Site: Redirect to logical url

How a user sends a message

  • User: Accesses the site
  • User: Click on the new message icon
  • Site: Open new message window and create new message tab on sidebar
  • User: Type the name of the user you would like to message in search field
  • Site: Auto fill user input with logical suggestions
  • User: Select intended user
  • Site: Display selected user in message field and include friend information
  • User: Type in message field
  • Site: Spell check user input
  • User: Press return
  • Site: Display user submitted input in message window
